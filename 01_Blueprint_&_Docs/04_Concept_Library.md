# Concept Library

## Purpose

The Concept Library is a permanent reference documenting the musical, educational, arranging, visual, and cultural concepts used throughout the Philippine Marching Percussion Fusion Project.

Rather than scattering explanations throughout multiple documents, concepts are defined once here and referenced elsewhere.

Each concept should include:

- Definition
- Purpose
- Application within this project
- Practical implementation
- Relationship to other concepts

---

# Current Concepts

## 1. Syncopation in the Philippine Context

### Definition

Syncopation is the deliberate placement of accents on weak beats or offbeats rather than on expected strong beats.

### Purpose

Syncopation provides forward momentum, excitement, and rhythmic tension.

### Application

Philippine festival drumming naturally emphasizes offbeat bass drum rhythms that create an energetic street groove.

This rhythmic identity forms the foundation upon which the remainder of the percussion writing is built.

### Relationship

Forms the rhythmic foundation for:

- Groove Construction
- Bass Drum Voicing
- Hybrid Rudiments
- Metric Modulation

---

## 2. Split Orchestration

### Definition

Split orchestration divides a single musical phrase among multiple performers, creating one continuous musical line shared across the ensemble.

### Purpose

Creates greater speed, musical range, and visual movement than a single performer can achieve alone.

### Application

For DCI tenor lines, split orchestration creates fluid multi-drum melodic movement.

For traditional Philippine single-tenor lines, players with differently pitched drums collectively create a melodic instrument through coordinated note distribution.

### Relationship

Supports:

- Tenor Writing
- Voice Leading
- Ensemble Coordination
- Visual Design

---

## 3. Marching Percussion Notation Strategy

### Definition

A standardized notation system designed for accurate playback, clear performer interpretation, and future expansion.

### Application

MuseScore Studio 4 will serve as the project's official notation platform.

Notation should emphasize readability while maintaining compatibility with Muse Drumline playback.

### Relationship

Supports every composition and educational document within the project.
