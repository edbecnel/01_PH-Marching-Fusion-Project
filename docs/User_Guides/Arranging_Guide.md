# Arranging Guide

> **Status:** Draft
> **Owner:** Philippine Marching Percussion Fusion Project
> **Applies To:** Arrangers and section leaders
> **Last Reviewed:** 2026-07-13
> **Review Frequency:** Quarterly

## Purpose

This guide explains how to apply concepts defined in the [Identity Framework](../Specifications/Philippine_Marching_Percussion_Identity_Framework.md), [Design Principles](../Specifications/Design_Principles.md), and [Concept Library](../Reference/Concept_Library.md). It does not redefine project identity.

Educators new to the project should read the [Instructor Introduction](Instructor_Introduction.md) first.

## How to Use This Guide

1. Read the Identity Framework and apply the [Identity Preservation Checklist](../Specifications/Philippine_Marching_Percussion_Identity_Framework.md#appendix-identity-preservation-checklist).
2. Consult relevant Concept Library entries.
3. Follow [Notation Standards](../Specifications/Notation_Standards.md) when writing parts.
4. Validate arrangements against the future [Philippine Percussion Sound Model](../Reference/Philippine_Percussion_Sound_Model.md).

## Arranging Responsibilities

This guide will explain how to:

- translate indigenous rhythms into battery writing
- layer traditional grooves as the harmonic and rhythmic foundation
- apply bass-drum philosophy rooted in festival momentum
- translate hand-drum articulation into battery textures
- preserve Philippine rhythmic identity while increasing musical sophistication
- use the three-layer method (soul, precision, contrast) from the [Core Blueprint](../Specifications/Core_Blueprint.md)
- adapt writing for available local instrumentation

## Three-Layer Method Summary

| Layer | Role | Source |
|---|---|---|
| Layer A — The Soul | Native groove, bass momentum, cymbal energy | Philippine festival and Drum & Lyre traditions |
| Layer B — The Precision | Rudimental and battery vocabulary over the groove | Selected DCI technique |
| Layer C — The Contrast | Showmanship, breakdowns, metric shifts | Selected HBCU and contemporary influence |
